# AI Sentiment Analysis

This is a Sentiment Analysis Web Application built using Python, NLP, Machine Learning and Streamlit.

## Features
- Predicts Positive and Negative Sentiments
- Uses TF-IDF Vectorization
- Uses Logistic Regression Model
- Interactive Streamlit Web Interface

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Streamlit
- NLP

## Dataset
IMDb 50K Movie Reviews Dataset
